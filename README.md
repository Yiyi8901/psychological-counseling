\# psychological counseling 心理咨询对话系统

\## 目录说明

1\. backend：后端完整工程

\- src：业务接口核心源码

\- \*.sql / \*.py：数据库创建、会话迁移、接口测试脚本

\- pom.xml：Maven后端项目依赖配置

\- .vscode、target：编辑器配置与项目编译输出文件



2\. frontend：前端UI设计模块

\- ui\_screenshot：系统聊天界面原型截图，包含主对话窗口、左侧历史对话侧边栏完整布局



\## 后续迭代计划

使用React+Tailwind CSS开发可交互前端页面，实现三大心理技能：

1\. 助眠引导

2\. 认知重构（CBT Reframing）

3\. 情绪急救
